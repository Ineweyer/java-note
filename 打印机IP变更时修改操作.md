# 打印机IP变更时修改操作

分为两步：

第一步修改打印机的ip，第二步修改ip更改软件的中的打印机ip，两个步骤缺一不可。下面以打印机ip为219.216.64.115进行举例。

## 1. 修改打印机IP地址

1.1 从设置里面打开已经安装好的打印机，找到并点击其**管理**选项，在win10下是找到打印机后点击一下就能够展开操作部分。
